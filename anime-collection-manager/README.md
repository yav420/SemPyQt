# Аниме-менеджер

Программа для ведения учета просмотренных аниме на PyQt6.

## Возможности
- Добавление, редактирование и удаление аниме
- Ведение отзывов и заметок о просмотренных аниме
- Загрузка постеров аниме
- Статистика просмотра (графики и диаграммы)
- Поиск по названию и студии
- Экспорт данных в CSV
- Современный интерфейс с вкладками

## Требования
- Python 3.8+
- PyQt6
- Matplotlib

## Установка
```bash
pip install PyQt6 matplotlib
```

## Запуск
```bash
python src/main.py
```

## Структура проекта
- `src/` - исходный код программы
- `qt/` - файлы интерфейса
- `anime_manager.db` - база данных

## Использование
1. Добавьте аниме через кнопку "Добавить аниме"
2. Заполните информацию: название, студия, жанр, статус, оценка
3. Отслеживайте прогресс просмотра
4. Используйте поиск для быстрого доступа
5. Смотрите статистику в разделе "Статистика"
